# IELTS Atelier Prototype

一个免安装即可运行的雅思学习平台前端原型。

## 运行方式

直接双击打开 `index.html`，或在当前目录启动任意静态文件服务后访问。

## 已覆盖的核心功能

- 邮箱注册 / 登录
- 注册后 6 步学习信息引导
- 基于目标分数、考试日期、已有资料、空闲时间与模块选择生成学习任务
- “你的计划已生成”过渡页
- 首页模块卡片与今天总任务区
- 任务详情页自动计时、备注、完成反馈
- 学习记录页
- 打卡页
- 授权 / 共享页
- 个人设置页
- 完整学习计划页
- Apple Calendar `.ics` 导出 / 导入原型
- 阅读 / 听力高保真机考练习页原型
- 自定义任务与系统任务并存

## 说明

- 数据使用浏览器 `localStorage` 保存，适合原型演示。
- 授权系统需要在同一浏览器里注册多个账号后切换登录进行体验。
